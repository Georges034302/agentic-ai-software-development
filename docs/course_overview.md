# Course Overview

> **Agentic AI Software Development Focus**
>
> This course is dedicated to the engineering of agentic AI systems for software development—autonomous, goal-driven agents that plan, act, and adapt in real-world engineering workflows. All modules, docs, and examples are focused on agentic architectures for software engineering, not general AI topics.

A tree view of the **Agentic AI Software Development** course structure. Each module is a separate file in [`modules/`](../modules); the full clickable index is in [`docs/index.md`](index.md).
