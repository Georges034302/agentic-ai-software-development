# Glossary — Agentic AI Software Development

**Agentic AI**: Autonomous, goal-driven AI systems that plan, act, and adapt in real-world workflows.

**MCP (Model Context Protocol)**: A protocol for integrating AI agents with developer tools, APIs, and workflows.

**RAG (Retrieval-Augmented Generation)**: A technique that combines retrieval of relevant documents or code with generative AI models for improved accuracy and context.

**Guardrails**: Mechanisms to ensure safe, compliant, and reliable AI behavior.

**Supervisor/Worker Pattern**: A multi-agent workflow where a supervisor agent delegates tasks to worker agents.

**AI-in-the-Loop**: Software engineering workflows where AI agents assist or automate parts of the SDLC (Software Development Lifecycle).

**Backend**: The AI model or platform powering the agent (e.g., Anthropic Claude, AWS Kiro).

**Observability**: Monitoring and logging agent actions for debugging, compliance, and improvement.
