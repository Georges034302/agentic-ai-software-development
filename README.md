# Agentic AI Software Development

> **Course Overview:**
This repository provides a hands-on, modular curriculum for mastering agentic AI techniques in software development. It covers foundational concepts, practical engineering, tool integration, multi-agent workflows, retrieval-augmented generation (RAG), and productionization, all tailored for modern software engineering teams.

---

## 📚 Lessons & Modules

1. **Introduction to Agentic AI for Software Development**
   - What is agentic AI?
   - Why it matters for software engineering
   - Key concepts and terminology

2. **Prompt Engineering for Software Agents**
   - Crafting effective prompts for coding tasks
   - Context design and prompt templates
   - Testing and iterating prompts

3. **Connecting Agents to Developer Tools**
   - Integrating with IDEs, CLIs, and APIs
   - Model Context Protocol (MCP) basics
   - Tool registry and invocation patterns

4. **Stateful and Guardrailed Agents**
   - Memory, context, and session management
   - Implementing guardrails for safe code generation
   - Logging and observability

5. **Multi-Agent Workflows in Software Projects**
   - Supervisor/worker agent patterns
   - Communication and coordination
   - Workflow orchestration

6. **Agentic RAG for Codebases**
   - Setting up vector databases for code
   - Retrieval-augmented generation for documentation and code
   - Governance and citation enforcement

7. **AI-Augmented Software Engineering**
   - Code reading, patch proposal, and test automation
   - Simulating pull requests and code reviews
   - Best practices for AI-in-the-loop development

8. **Productionizing Agentic Systems**
   - Observability, logging, and policy enforcement
   - Error handling and scaling
   - Deployment strategies

## 🤖 Agent Backend Options

This curriculum supports multiple agent backends for hands-on labs and code samples:

- **Anthropic Claude**: Advanced LLM agent capabilities, accessible via Anthropic's API.
- **AWS Kiro**: Enterprise-grade AI agent platform, integrated with AWS services.

You can choose your preferred backend for each lesson or lab. Configuration instructions are provided in the relevant modules and labs.
