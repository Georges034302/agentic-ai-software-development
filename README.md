
# Agentic AI Software Development

A structured course on engineering modern AI-powered software systems using **agentic architectures** and **enterprise AI workflows**. The curriculum is inspired by industry concepts and tooling associated with Anthropic Claude, AWS Kiro, MCP, and contemporary agentic AI platforms for software engineering teams.

---

## Project Overview

This repository contains an 8-module curriculum that takes learners from foundational concepts to production-grade agentic systems. Across the modules you will learn how to:

- Build autonomous AI agents for software development
- Engineer effective prompts and context for coding agents
- Integrate agents with IDEs, CLIs, APIs, and code repositories
- Implement stateful, guardrailed, and auditable agents
- Design multi-agent workflows for real-world projects
- Develop retrieval-augmented generation (RAG) for codebases
- Apply AI to the software development lifecycle (SDLC)
- Deploy, scale, and govern production agentic systems

---

## Course Architecture

The curriculum is organized as a progressive 8-module learning path. Each module builds on the previous, moving from foundational concepts through to production deployment and governance.

```
Foundations ─▶ Prompting ─▶ Tool Integration ─▶ Stateful Agents ─▶ Multi-Agent ─▶ RAG ─▶ AI SDLC ─▶ Production
```

## Agent Backend Options

This curriculum supports multiple agent backends for hands-on labs and code samples:

- **Anthropic Claude**: Advanced LLM agent capabilities, accessible via Anthropic's API.
- **AWS Kiro**: Enterprise-grade AI agent platform, integrated with AWS services.

You can choose your preferred backend for each lesson or lab. Configuration instructions are provided in the relevant modules and labs.
