# Lab 7: Agentic Software Engineering

## Objective
Simulate AI-augmented software engineering workflows: patching, testing, PRs, and reviews.

## Steps
1. Use agents to propose code patches and run tests.
2. Simulate pull requests, reviews, and merges.
3. Apply best practices for AI-in-the-loop development.

## Deliverables
- Example agent-driven SDLC
- Logs of PRs, reviews, and merges
