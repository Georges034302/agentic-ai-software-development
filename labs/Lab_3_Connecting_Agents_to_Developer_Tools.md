# Lab 3: Connect Agent to Tools via MCP

## Objective
Integrate an agent with developer tools using the Model Context Protocol (MCP).

## Steps
1. Set up an MCP server and client.
2. Expose developer tools (e.g., CLI, API) to the agent.
3. Demonstrate agent invocation of tools via MCP.

## Deliverables
- MCP integration code
- Example agent-tool interaction
