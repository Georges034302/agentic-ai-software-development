# Lab 1: Minimal Agentic Loop

## Objective
Build a minimal agentic loop for software development: plan, act, observe, and adapt.

## Steps
1. Define a simple agent goal (e.g., "Refactor a function").
2. Implement the agent loop: plan → act → observe → adapt.
3. Log each step and outcome.

## Deliverables
- Code for the agentic loop
- Example run and log
